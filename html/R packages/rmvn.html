Description

produce Gaussian Markov fields.

Usage

rmvn(n, mu = 0, L)

Arguments

n

sample size of random numbers.

mu

mean of Gaussian Markov fields.

L

cholesky factorization of covariance matrix of Gaussian Markov fields.
